Pertama buka terminal pada editor yang dimiliki.
Ketik "npm i" untuk melakukan instalasi semua library yang dipasang.
Lalu ketik "npm start" untuk melakukan proses compile.
Pada bagian home dapat dilihat ada nabar, card, dan footer

Navbar berfungsi untuk routing pindah halaman
Card berfungsi sebagai tempat ditaruhnya data Nama, Tanggal lahir, dll.
Footer berisi informasi lainnya seperti facebook,instagram, dll.

Dalam proses pengambilan data akan muncul tulisan Loading...
Jika data berhasil diambil makan akan muncul Card tempat data ditaruh.
Jika data tidak berhasil diambil maka akan muncul tulisan "Error".
